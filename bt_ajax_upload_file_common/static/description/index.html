Ajax Upload File Common

This Module enabling upload file using ajax on Website.

Key Highlights

  • Call template for enable upload file on website.
  • Assign value to variable for store uploaded file to specific object/model.
  • Overview
  • Screenshots

Technical Guide

  • 1. For editable upload file mode call following block and set vaule to variable.

    • <t t-call="banas_upload_file_base.upload_file">
     <t t-set="file_input_name" t-value="'attachments'"/>
     <t t-set="res_model" t-value="'crm.lead'"/>
     <t t-set="res_id" t-value=""/>
     <t t-set="max_file_size" t-value="10"/>
     <t t-set="allow_multi_file" t-value="'true'"/>
     <t t-set="show_delete" t-value="'true'"/>
     <t t-set="show_download" t-value="'true'"/>
    </t>

  • 2. For readonly upload file mode call following block and set vaule to variable.

    • <t t-call="banas_upload_file_base.upload_file">
     <t t-set="file_input_name" t-value="'attachments'"/>
     <t t-set="res_model" t-value="'crm.lead'"/>
     <t t-set="res_id" t-value=""/>
     <t t-set="max_file_size" t-value="10"/>
     <t t-set="allow_multi_file" t-value="'true'"/>
     <t t-set="show_delete" t-value="'false'"/>
     <t t-set="show_download" t-value="'false'"/>
    </t>

  • 3. Once called above code in templates. When do form submit controller will receive file_ids variable in post of ir.attachment object ids where you can do computation or if you set res_id it will auto attach file to record belong to res_model and res_id.
